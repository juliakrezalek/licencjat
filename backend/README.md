# backend

ten folder zawiera backend dla projektu badawczego z wykorzystaniem chatbotów.

## zawartość
- `wytwor.py` – główny plik uruchamiający
- `chatbot1_ekspert.md`, `chatbot2.md`, `chatbot3_halucynujacy.md` – prompty i konfiguracje dla różnych wersji chatbota
- `pyproject.toml`, `uv.lock` – pliki zależności

## ważne
- pliki `.env` oraz `credentials.json` są ignorowane przez `.gitignore`
- do uruchomienia wymagany jest klucz openai (trzymany lokalnie w replit)
